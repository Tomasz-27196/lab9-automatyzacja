## Code review

W ramach przeglądu kodu wykonano następujące poprawki:
- dodano nagłówek opisowy na początku skryptu,
- ujednolicono definicję ścieżek do katalogów roboczych (zmienne bazowe),
- dodano obsługę błędów i czytelne komunikaty,
- ustandaryzowano format logów oraz raportów,
- wykonano ponowne testy po wprowadzeniu zmian.
